CSV Format Information

Please ensure your CSV file follows this format:

Required Columns:
  • date: Date in YYYY-MM-DD format
  • time: Time in HH:MM:SS format
  • ambient_temperature: Ambient temperature in °C
  • water_temperature: Water temperature in °C
  • glass_temperature: Glass temperature in °C
  • solar_radiation: Solar radiation in W/m²
Optional Columns:
  • wind_speed: Wind speed in m/s
  • humidity: Humidity in %
  • efficiency: Efficiency in %
  • freshwater_quantity: Freshwater produced in ml
Example:
date,time,ambient_temperature,water_temperature,glass_temperature,solar_radiation,wind_speed,humidity,efficiency,freshwater_quantity
2023-06-01,10:00:00,30.5,45.2,35.8,750.5,2.5,65.3,42.3,350.2
2023-06-01,11:00:00,32.1,47.5,38.2,820.0,3.1,60.2,45.1,385.7
If efficiency and freshwater_quantity are not provided, the system will predict these values using the active model.
